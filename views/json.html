JSON: JavaScript Object Notation

¿Qué es 'JSON' en JavaScript?

Es un formato basado en texto estándar para representar datos estructurados en la sintaxis de objetos de JavaScript. Es comúnmente utilizado para transmitir datos en aplicaciones web.
Por ejemplo: enviar algunos datos desde el servidor al cliente, así estos datos pueden ser mostrados en páginas web, o viceversa.


Un archivo JSON suele contener mucha información almacenada, ej. sintaxis:

{
  "string": "Luis",
  "number": 25,
  "boolean": true,
  "array": ["programar", "jugar"],
  "object": {
    "instagram": "@jabalera_luis",
    "email": "jabaleraluisangel@gmail.com"
  },
  "null": null
}

Sin embargo, su contenido puede ser simplemente un Array, un Number, un String, un Boolean o incluso un Null, lo más habitual es que parta siendo un Object o un Array.


Métodos de JSON

JSON.parse()

Analiza una cadena de texto y lo convierte a un objeto válido para JavaScript.


JSON.stringify()

Hace lo contrario a 'parse', analiza un objeto o un valor válido para JavaSript y lo convierte a cadena de texto.
